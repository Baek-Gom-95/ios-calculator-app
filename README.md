## 10진수 계산기 고려사항
### 연산자 버튼을 클릭 할 때 마다 화면 출력되는 숫자가 다름(연산이 다름)
```
2+3+ -> 5
2x3+ -> 6
2x3x -> 6
2+3x5+ -> 17
2+3x5x -> 15
```
### 3자리 수 마다 콤마(,)를 화면에 표시
```
1,000
123,456,789
42.233456
```
### 정수부와 소수부의 모든 숫자의 개수가 9자리를 넘으면 안됨.
### 마지막 입력이 연산자이고 그 다음 입력이 `=` 일때 처리
